#Markdown links

É uma biblioteca consegue retirar de um documento markdown os links .

##Como instalar:

```$ npm install -g erickalemos/SAP003-md-links```

Para executar esta biblioteca você deverá ter o node e npm instalados.

##Como utilizar:

```$ md-links exemplo.md```
